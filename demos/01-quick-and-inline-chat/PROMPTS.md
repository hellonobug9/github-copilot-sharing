# 01 – Quick Chat & Inline Chat (Simple Demo)

## Rule of thumb

- **Inline Chat** → dùng khi muốn Copilot **giải thích / sửa / refactor một đoạn code cụ thể**  
- **Quick Chat** → dùng khi muốn **hỏi nhanh về kiến thức, best practices, thư viện, cách làm**

Câu ngắn để nói trong buổi sharing:

> Câu hỏi dính tới đoạn code cụ thể → Inline Chat.  
> Câu hỏi về kiến thức / best practice / thư viện → Quick Chat.
